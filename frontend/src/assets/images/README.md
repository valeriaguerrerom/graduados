# Instrucciones para agregar el logo

## Pasos para configurar el logo de la Universidad Mariana:

1. **Descargar el logo oficial** de la Universidad Mariana en formato PNG
   - Preferiblemente en alta resolución
   - Fondo transparente si es posible

2. **Guardar el archivo** en esta carpeta con el nombre:
   ```
   logo-umariana.png
   ```

3. **El logo se mostrará automáticamente** en:
   - Navbar (esquina superior izquierda)
   - Footer (sección de información)
   - Página de Login (encabezado)

## Formatos recomendados:
- **Formato**: PNG (con transparencia)
- **Tamaño**: Mínimo 200x200px
- **Nombre del archivo**: `logo-umariana.png`

## Si no tienes el logo:
Puedes descargarlo desde:
- Sitio web oficial de la Universidad Mariana
- Redes sociales oficiales
- O solicítalo a la oficina de comunicaciones

Una vez que agregues el archivo `logo-umariana.png` en esta carpeta, el logo aparecerá automáticamente en toda la aplicación.
